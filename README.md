# Poster


Simple reddit-esque social media like website (hopefully)


## NOTES

### Models

- User
    - username
        - Login name
    - display name
    - password
- Post
    - title
    - content
    - author
    - date
    - comments
        - List of comments under it
    - Upvotes
        - List of users who upvoted
    - Downvotes
        - List of users who downvoted
- Comment
    - content
    - author
    - date
    - comments
        - List of comments under it
    - Upvotes
        - List of users who upvoted
    - Downvotes
        - List of users who downvoted

### Views

- Home
    - List of posts
    - Top bar
        - Search
        - (if logged in) Logout
        - (if not logged in) Login
        - (if not logged in) Sign up
    - Post button
- Login page
- Sign up page
- Create post page
